# Proxy Scrapper | http, https, socks4, socks5

Get free proxy, extract & check. Available proxy: http, https, socks4, socks5

## Installation

Tested on : 
- python `>=3.5`

Installation
```bash
pip install -r requirements.txt
```

## Run App

Get free proxy
```bash
python get_proxy.py
```

Check proxy
```bash
python check_proxy.py
```
